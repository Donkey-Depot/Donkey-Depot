# Donkey Depot Website

The official launch website for Donkey Depot, showcasing our vision for a circular economy in Juja Then the world and introducing the next generation of the EcoLoop platform.

## Features
- Interactive roadmap and project updates
- Company vision and team information
- Product and partnership previews
- EcoLoop app rebuild showcase
- Newsletter and beta waitlist signup
- Contact and community engagement forms

## Tech Stack
- HTML, CSS, and JavaScript
- Vercel for hosting and deployment

## Getting Started
Clone the repository and open `src/index.html` in your browser, or deploy the project using Vercel for the full experience.
